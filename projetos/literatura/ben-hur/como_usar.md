# Resumo do script ben_hur_runner.py:

| Feature               | Detalhe                                 |
| --------------------- | --------------------------------------- |
| Modo fire-and-forget  |  Dispara criação e segue (~2min/cena) |
| --dry-run             |  Mostra plano sem executar              |
| --max-scenes N        |  Limita número de cenas                |
| --livro N             |  Filtra por livro (1-8)                 |
| --download            |  Baixa áudios já processados          |
| --no-wait             |  Sem intervalo (testes)Prompts          |  81/81 customizados COF |
| Source targeting      |  Sempre foca no source do Ben-Hur       |
| Metadata              |  Salva em audios/metadata.json          |
| Logs                  |  Salva em logs/benhur_session_*.json    |
| Ctrl+C                |  Graceful shutdown                      |
