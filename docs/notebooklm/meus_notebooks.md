# Meus Notebooks do NotebookLM

Este arquivo serve como um diretório dos seus notebooks para uso com o servidor MCP no Claude.

**Como usar:**
1.  **Encontre o ID**: Abra o notebook no navegador. O ID é a parte final da URL (após `notebook/`).
    *   Exemplo: `https://notebooklm.google.com/notebook/1a2b3c4d-5e6f-7g8h` -> ID: `1a2b3c4d-5e6f-7g8h`
2.  **Liste abaixo**: Adicione seus notebooks importantes aqui.
3.  **No Claude**: Diga "Defina meu notebook padrão para o ID do Projeto X".
